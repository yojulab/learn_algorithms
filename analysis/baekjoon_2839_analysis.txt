from https://www.acmicpc.net/problem/2839

- 문제 파악
N킬로그램을 배달
봉지는 3킬로그램 봉지와 5킬로그램 봉지
최대한 적은 봉지를 들고 가려고 한다
18킬로그램 설탕 배달 할 때, 3킬로그램 봉지 6개도 되지만, 5킬로그램 3개와 3킬로그램 1개를 배달
-> 18 = 5a + 3b 로 구할 때 a 개수가 최대한 많은 것 선택


- 주요 단어 영문 이름 선정 
N킬로그램 : n_kg, n_weight
배달 : delivery
봉지 : bag
최대한 적은 봉지 : minimum_bag

- 테스트 케이스
18  -> 4
4   -> -1
6   -> 2
9   -> 3
11  -> 3

- 프로그래밍 순서
배달 총 킬로그래 받기(3 ≤ N ≤ 5000)
배달 최소 봉지 개수 구하기.
배달 최소 봉지 개수 표시(정확하게 N킬로그램을 만들 수 없다면 -1을 출력)
+ try-on

- 실패 시 원인 기록

- 느낀 점 or 배운 점
